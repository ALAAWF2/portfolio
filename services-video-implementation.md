# دمج فيديوهات الخدمات

## الهدف

تحويل قسم الخدمات إلى معرض احترافي من 12 بطاقة فيديو ثنائية اللغة، سريع ومتجاوب وقابل للوصول.

## المهام

- [x] نسخ ملفات MP4 إلى `assets/videos/services/` → التحقق: وجود 12 ملفًا بالأسماء المعتمدة.
- [x] استبدال بطاقات الخدمات في `index.html` بقائمة الـ12 → التحقق: 12 بطاقة و12 مصدر فيديو.
- [x] تحديث نصوص AR/EN وخيارات نموذج التواصل في `script.js` → التحقق: لا مفاتيح ترجمة ناقصة في اللغتين.
- [x] إضافة منطق تشغيل وإيقاف الفيديو → التحقق: لا يعمل أكثر من فيديو واحد، والزر يعمل بلوحة المفاتيح.
- [x] إضافة تصميم Live Operations Gallery في `style.css` → التحقق: 3/2/1 أعمدة وحالات focus/reduced-motion سليمة.
- [x] تحديث بيانات SEO المنظمة وقائمة `knowsAbout` → التحقق: الخدمات الجديدة ممثلة في الصفحة والـJSON-LD.
- [x] اختبار الموقع محليًا بالعربية والإنجليزية وعلى سطح المكتب والجوال → التحقق: لا أخطاء console ولا ملفات 404.

## تمّ عند

- [x] جميع الخدمات الـ12 مرئية ومطابقة لفيديوهاتها.
- [x] الفيديو متجاوب وموفّر للبيانات ويحترم إعدادات الحركة.
- [x] نموذج التواصل والترجمة يعملان في اللغتين.
