# حزمة تحسين Portfolio

## الهدف

إعادة بناء الصفحة كـPortfolio بسيط ومقنع، مع عرض فيديو خدمات كبير ودعم كامل للثيمين واللغتين.

## المهام

- [x] إعادة ترتيب الصفحة إلى 6 محطات واضحة → التحقق: اختفاء الأقسام المتكررة من مسار القراءة.
- [x] بناء Hero ومؤشرات ثقة مختصرة → التحقق: القيمة وCTA ظاهران ضمن الشاشة الأولى.
- [x] تحويل المشاريع إلى عرض تحريري مختار → التحقق: المشاريع السبعة ما زالت قابلة للوصول.
- [x] تحويل الخدمات إلى Showreel بفيديو واحد كبير → التحقق: 12 خدمة تبدّل الفيديو والنص بلا تحميل جماعي.
- [x] دمج النبذة والقطاعات وطريقة العمل → التحقق: المحتوى الأساسي محفوظ ضمن قسم واحد.
- [x] تحديث CSS للثيمين والاستجابة → التحقق: Desktop/Mobile وLight/Dark بلا overflow أو contrast ضعيف.
- [x] تحديث JavaScript والترجمة والتفاعل → التحقق: اللغة والثيم والفيديو والنموذج تعمل.
- [x] اختبار نهائي في المتصفح → التحقق: لا console errors أو 404 وجميع السيناريوهات تمر.

## تمّ عند

- [x] الفيديو هو العنصر البصري الأساسي في الخدمات وليس بطاقة صغيرة.
- [x] الصفحة أقصر وأوضح وتعرض الأعمال والخدمات بلا تكرار.
- [x] العربية والإنجليزية والثيمان والجوال تعمل بصورة متكافئة.
