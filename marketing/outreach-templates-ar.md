# قوالب رسائل التواصل — WafaieTech

> **قواعد ذهبية قبل ما ترسل:**
> - لا ترسل نسخة-لصق حرفية. غيّر أول سطر حسب الشخص/الشركة (اسم المحل، شي لاحظته).
> - الرسالة الأولى هدفها "نعم بحب أشوف"، مو البيع. خليها قصيرة.
> - اسأل سؤال بآخر الرسالة حتى يسهل الرد.
> - تابع مرتين على الأقل قبل ما تعتبر الشخص "غير مهتم".

---

## 1) واتساب — تواصل بارد مع سلسلة تجزئة / صاحب محلات متعددة

```
السلام عليكم أستاذ [الاسم]، معك علاء وفائي.
بشتغل بأنظمة تجمع مبيعات الفروع والمخزون والتقارير في لوحة واحدة للإدارة.

لاحظت إن [اسم الشركة] عندها كذا فرع — عادةً بهالحجم بيضيع وقت كبير
بتجميع تقارير المبيعات يدوياً من الفروع.

عندي ديمو دقيقتين بيوريك كيف بتنحل. حابب أبعتهولك بدون أي التزام؟
```

## 2) واتساب — تواصل بارد مع تاجر متجر إلكتروني (سلة / ووكومرس)

```
مساء الخير أستاذ [الاسم]، معك علاء وفائي.
متخصص بربط المتاجر الإلكترونية (سلة/ووكومرس) مع المخزون والـ ERP.

من أكتر المشاكل اللي بشوفها عند المتاجر: فرق بالمخزون وإلغاء طلبات
لأن الكميات ما بتتحدّث لحظياً.

نفّذت محرك مزامنة قلّل هالمشكلة بشكل كبير وسرّع الطلبات ~60%.
حابب أوريك كيف بشتغل على متجرك؟
```

## 3) واتساب — طلب إحالة من معرفة (أسرع طريق لأول عميل)

```
مرحبا [الاسم]، كيفك؟
بلّشت أشتغل مستقل بأنظمة الأتمتة والتقارير لشركات التجزئة والمتاجر.

مو طالب منك شي — بس إذا بيخطر ببالك أي حدا عنده محلات أو متجر إلكتروني
وبيتذمّر من التقارير اليدوية أو فوضى المخزون، وصّلني فيه ولك جزيل الشكر 🙏
```

## 4) واتساب — رسالة متابعة (بعد يومين-ثلاثة بدون رد)

```
أستاذ [الاسم]، بس أتأكد إنه رسالتي وصلتك 🙏
جهّزت الديمو القصير، أبعتهولك؟ ولو مو الوقت المناسب، عادي تماماً خبّرني.
```

## 5) إنستغرام — رسالة DM لبراند تجزئة / مطعم

```
أهلاً 👋 متابع شغلكم وحلو التطور.
أنا علاء، بساعد محلات التجزئة والمطاعم يأتمتوا تقاريرهم ويربطوا
مبيعات الفروع بلوحة وحدة بدل التجميع اليدوي.

في عندكم عدة فروع؟ بحب أبعتلكم مثال سريع كيف بيوفّر وقت الإدارة.
```

> نصيحة إنستغرام: تفاعل مع منشوراتهم كم يوم قبل الـ DM، وخلي بايو حسابك واضح إنك بتقدّم أنظمة أتمتة للتجزئة + لينك الموقع.

## 6) LinkedIn — طلب اتصال + رسالة بعد القبول

**مع طلب الاتصال (قصير):**
```
أهلاً أستاذ [الاسم]، أتابع مجال التجزئة وأنظمة العمليات.
يسعدني أضيفك للتواصل المهني.
```

**بعد قبول الاتصال:**
```
شكراً على القبول أستاذ [الاسم].
أنا متخصص ببناء لوحات BI وأتمتة التقارير وربط أنظمة الـ ERP/POS
لشركات التجزئة متعددة الفروع. من مشاريعي ربط 52 فرعاً بلوحة لحظية واحدة.

إذا فريقكم بيقضي وقت كبير بتجميع تقارير الفروع يدوياً،
يسعدني أشاركك مثال قصير كيف يمكن أتمتته. مناسب أبعته؟
```

## 7) منصات الفريلانس (مستقل / خمسات / Upwork)

> خصّص أول سطرين لتفاصيل الطلب نفسه — هاد اللي بيفرّق عن المتقدمين الباقيين.

```
السلام عليكم،
قرأت طلبك بخصوص [اذكر تفصيل محدد من الإعلان — مثلاً: ربط متجر سلة بالمخزون].

هاد بالضبط اختصاصي: ببني أنظمة أتمتة ولوحات بيانات وربط أنظمة لشركات التجزئة.
أمثلة على شغلي:
• محرك مزامنة ربط متاجر إلكترونية مع المخزون والـ ERP (سرّع الطلبات ~60%).
• منصة تحليلات ربطت 52 فرعاً بلوحة لحظية واحدة.
• نظام أتمتة قلّص تجهيز التقارير من 4 ساعات لدقائق.

التقنيات: Python, PostgreSQL, React, Supabase, ربط APIs و ERP.

أقدر أبدأ بنطاق صغير محدد حتى تشوف النتيجة قبل ما نكمّل.
ممكن نحدد مكالمة قصيرة نراجع تفاصيل طلبك؟

أعمالي: alaa-wafaie.com
علاء
```
