# Precision Studio Implementation

## Goal

إعادة بناء معرض الأعمال بواجهة تحريرية تقنية حديثة وسريعة تعرض المشاريع والخدمات بوضوح على جميع الشاشات.

## Tasks

- [x] إعادة كتابة `index.html` بهيكل Hero، أعمال، Showreel، خبرة، وتواصل دلالي — Verify: المحتوى والروابط واللغتان موجودة.
- [x] إعادة كتابة `style.css` بنظام ألوان وتخطيط Precision Studio — Verify: الوضعان الفاتح والغامق يعملان دون overflow.
- [x] إعادة كتابة `script.js` للتبديل اللغوي والثيم والفيديو والقائمة والنموذج — Verify: `node --check script.js` ينجح.
- [x] تحديث `sw.js` وأرقام cache-busting — Verify: الملفات الجديدة ضمن cache بإصدار جديد.
- [x] اختبار سطح المكتب والجوال بالعربية والإنجليزية — Verify: لقطات 1440px و390px بلا قص أو تداخل.
- [x] مراجعة الإتاحة والأداء وإرشادات الواجهة — Verify: لوحة المفاتيح، reduced motion، أبعاد الصور، وتحميل الفيديو عند الطلب.

## Done When

- [x] المشاريع والفيديوهات تظهر بحجم واضح وهوية متماسكة.
- [x] الثيم واللغة والتنقل والنموذج تعمل.
- [x] لا توجد أخطاء JavaScript أو روابط أصول مفقودة أو مشاكل تخطيط ملحوظة.
