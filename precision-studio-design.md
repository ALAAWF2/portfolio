# Precision Studio Redesign

## Understanding

- الموقع هو معرض أعمال وخدمات لعلاء وفائي في أنظمة الأعمال والأتمتة وتحليل البيانات.
- الجمهور الأساسي شركات التجزئة والتجارة الإلكترونية والمؤسسات متعددة الفروع.
- الهدف هو إثبات الخبرة سريعاً، عرض أثر المشاريع، ثم تحويل الزائر إلى تواصل مباشر.
- يجب الحفاظ على العربية والإنجليزية، واتجاهي RTL وLTR، والوضعين الفاتح والغامق.
- يحتوي الموقع على 6 صور مشاريع و12 فيديو خدمة ويجب تقديمها دون الإضرار بالأداء.
- الموقع ثابت ويعمل مباشرة من GitHub أو أي استضافة ملفات دون خطوة بناء.

## Assumptions

- لا توجد حاجة إلى إطار عمل أو Backend جديد.
- نموذج التواصل يستمر بالتحويل إلى WhatsApp إلى أن يضاف Formspree حقيقي.
- أرقام الأثر والمحتوى وروابط السيرة الذاتية الحالية معتمدة.
- الحركة تحسين بصري فقط ويجب أن تختفي عند تفعيل تقليل الحركة.

## Direction

**Precision Studio**: هوية تحريرية تقنية هادئة، بخلفية عاجية في الوضع الفاتح وفحمية في الوضع الغامق، مع أسود حبري وبرتقالي إشارة. يعتمد التصميم على عناوين كبيرة، شبكة غير متناظرة، مساحات فارغة محسوبة، وأرقام أقسام واضحة.

العنصر المميز هو **Signal Line**: خط برتقالي رفيع يظهر في العنوان والتنقل وعناوين الأقسام ويربط رحلة الزائر بصرياً دون زخرفة زائدة.

## Information Architecture

1. Hero مختصر مع قيمة واضحة ونداءين للفعل.
2. شريط أرقام يثبت الخبرة والأثر.
3. أعمال مختارة: مشروع رئيسي كبير ثم شبكة مشاريع ثانوية.
4. Showreel خدمة واحدة كبيرة مع فهرس للخدمات الـ12.
5. الخبرة والمنهجية في مساحة تحريرية مختصرة.
6. تواصل مباشر ونموذج يفتح رسالة WhatsApp جاهزة.

## Design System

- Display: IBM Plex Sans Arabic / Manrope عند الإنجليزية.
- Body: Noto Sans Arabic / Manrope.
- الألوان كلها CSS custom properties وOKLCH مع fallback مفهوم.
- المقياس المكاني مبني على `clamp()` وتباعد 8 نقاط.
- الحدود دقيقة، الزوايا متوسطة، والظلال محدودة للميديا فقط.
- الحركة تعتمد opacity وtransform، مدتها 180–600ms، مع `prefers-reduced-motion`.

## Technical Decisions

- HTML دلالي مع skip link وتسلسل عناوين سليم.
- CSS حديث مع Grid وContainer Queries وlogical properties.
- JavaScript معياري صغير دون تبعيات.
- فيديو واحد فقط في DOM، `preload="none"`، ويتغير مصدره بعد اختيار المستخدم.
- الصور أسفل الصفحة lazy-loaded وبأبعاد ثابتة لمنع CLS.
- حفظ الثيم واللغة في localStorage، مع احترام تفضيل النظام أول مرة.
- Progressive enhancement فقط؛ المحتوى الأساسي يبقى متاحاً عند تعطيل JavaScript.

## Risks

- الفيديوهات ثقيلة: لا تحميل مسبق، posters، وإيقاف الفيديو خارج المشاهدة.
- النص الإنجليزي أطول أحياناً: `text-wrap`, container queries، واختبار الاتجاهين.
- الجوال: عناصر لمس 48px وفهرس خدمات أفقي واضح.
- الحركة: تقليلها تلقائياً وعدم استخدامها لإخفاء محتوى أساسي.

## Decision Log

- اختير Precision Studio بدل Product Lab وModern Brutalist لأنه الأكثر توازناً بين التميز والثقة.
- بقي المشروع Static لتقليل التعقيد وتحسين سهولة النشر والصيانة.
- المشاريع تسبق الخدمات لأنها أقوى دليل على القيمة.
- تم اعتماد showreel واحد بدل 12 بطاقة فيديو لتقليل الازدحام والتحميل.
- البرتقالي لون إشارة فقط، وليس خلفية مهيمنة، للحفاظ على الرقي والبساطة.
